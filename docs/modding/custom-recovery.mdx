### Bugs
- Vibrator

### Install
1. Download [latest release](https://github.com/korzenroot/android_device_motorola_fogorow-twrp/releases/download/TeamWin/vendor_boot.img)
2. Reboot to [fastboot mode](../modes/fastboot)
3. Disable vbmeta (if partition not found - ignore)
    ```shell
    $ fastboot flash vbmeta vbmeta.img --disable-verity --disable-verification
    ```
4. Flash TWRP
    ```shell
    $ fastboot flash vendor_boot vendor_boot.img
    ```
